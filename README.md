# Epic Dotfiles

## Setup

### Quick Setup
```bash
mkdir ~/dotfiles
cd ~/dotfiles
git clone <repo link>

makefile all
```
### Advanced Setup

#### Install only wanted apps
You can install only the wanted packages
```bash
makefile minimal # minimal Setup
makefile raspberry # Setup a raspberry server

# optionnal bundle 
makefile gui 
makefile server 
makefile design 
makefile office 
makefile dev 
```
